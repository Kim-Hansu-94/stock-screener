# 백테스트 가이드 (눌림목 + 골드스탠다드)

> 두 전략의 과거 성과를 실제 주가 데이터로 검증하는 방법과 규칙을 정리한 문서입니다.
> 매번 규칙을 까먹지 않도록 여기 모아둡니다. "백테스트 문서 열어줘" 라고 하면 이 파일을 보여드립니다.

## 한눈에 보기

| 항목 | 눌림목 스크리너 | 골드스탠다드 (오늘의 추천) |
|---|---|---|
| 성격 | 단기 스윙 (며칠~몇 달) | 장기 바닥 반등 (무기한 홀드) |
| 매수 시점 | 신호 발생일 **종가** | 추천점수 임계값 도달일 **종가** |
| 매도 규칙 | 20 / 60 / 120 거래일 후 청산 | **+300%** 도달 시 익절 (손절 없음) |
| 시장 필터 | SPY < SMA200 구간 신호 제외 (약세장 회피) | 없음 (프로덕션 채점 로직 그대로) |
| 재진입 쿨다운 | 종목당 30일 | 종목당 30일 (점수 구간별 독립) |
| 벤치마크 | SPY 동기간 수익률 | SPY 3년 매수후보유 |

## 데이터 출처

- **DB(Supabase)가 아니라 `yfinance` 로 그때그때 다운로드**합니다. DB는 최신 스냅샷만 갖고 있어서 과거 재생이 안 되지만, yfinance는 몇 년치 OHLCV를 바로 받아올 수 있습니다.
- 유니버스: S&P 500 + NASDAQ-100 + Russell 3000 (약 3,000 종목).
- 기간: 최근 **3년** (`YEARS_BACK = 3`).
- 다운로드에 약 10~20분 걸립니다. 두 백테스트가 **같은 데이터를 공유**하므로 다운로드는 한 번만 합니다.

## 실행 방법

```bash
# 프로젝트 루트에서
python pipeline/research/backtest_both.py
```

- 스크립트: `pipeline/research/backtest_both.py`
- 파이프라인(매일 스크리닝)과 **무관한 독립 리서치 스크립트**입니다. 돌려도 DB/프론트엔드에 영향 없습니다.

## 눌림목 규칙 상세

기존 검증된 `backtest_pullback.py` 의 `run_backtest` / `summarize` 를 그대로 재사용합니다.

- **신호 조건**: SMA10/20/60 정배열 상승, SMA20 ≤ 종가 ≤ SMA10, RSI14 40~60, 거래량 감소.
- **진입**: 신호 발생일 종가.
- **청산**: 20 / 60 / 120 거래일 각각에 대해 별도 집계 → 해당 캘린더일 이후 첫 거래일 종가.
- **약세장 필터**: 신호일에 SPY가 SMA200 아래면 그 신호는 버림.
- **쿨다운**: 같은 종목은 30일 안에 재진입 안 함.
- 출력: `stock_return_pct`, `bench_return_pct`(SPY), `excess_return_pct`(초과수익).

## 골드스탠다드 규칙 상세

실제 파이프라인 채점 함수 `pattern_discovery._score_candidate` 를 **그대로 import** 해서 매일 재생(walk-forward)합니다. 별도 재구현이 아니라 프로덕션과 동일한 점수입니다.

- **워크포워드**: 각 거래일 t 에서 직전 약 1년(`LOOKBACK_ROWS = 260`) 창만 넣어 채점 → 미래를 안 봄(lookahead 없음).
- **유동성 필터**: 일평균 거래대금 ≥ `MIN_DOLLAR_VOL`(30만 달러). 프로덕션 `compute_pattern_matches` 와 동일하게 채점 전에 먼저 거릅니다.
- **점수 구성** (0~1, 상한 1.0): 52주 고점 대비 하락률(0.3) + 저점 이탈 후 경과(0.4) + 거래량 유지율(0.3) + VCP 압축 보너스(0.1) + 이평 정배열 보너스(0.1).
- **매수**: 점수가 임계값을 넘는 첫날 종가.
- **점수 구간 3종**: 100% / 95% / 90% 를 각각 따로 집계 (`GS_THRESHOLDS = [1.00, 0.95, 0.90]`).
- **보유**: 무기한. 중간에 얼마가 빠지든 **손절 안 함**.
- **매도**: 진입가의 4배(**+300%**, `SELL_TARGET = 3.0`) 최초 도달일에 익절. 3년 끝까지 못 넘으면 마지막 종가로 평가(미청산).
- **쿨다운**: 같은 종목 30일. 점수 구간마다 독립 시계.

## 주요 상수 (스크립트 상단에서 조정 가능)

```
YEARS_BACK       = 3      # 백테스트 기간(년)
LOOKBACK_ROWS    = 260    # 골드스탠다드 채점 창(거래일 ≈ 1년)
SELL_TARGET      = 3.0    # 골드스탠다드 익절 목표(+300%)
GS_COOLDOWN_DAYS = 30     # 골드스탠다드 재진입 쿨다운(일)
GS_THRESHOLDS    = [1.00, 0.95, 0.90]  # 점수 구간
HOLDING_DAYS     = [20, 60, 120]       # 눌림목 청산 시점(거래일)
```
