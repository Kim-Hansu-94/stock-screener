작업 완료 후 항상 완료 신호를 주되, "더 시킬 일 있으세요?" 같은 표현은 쓰지 말 것. 기분 나쁘게 들린다고 함.

**Why:** 유저가 터미널을 못 보기 때문에 완료 신호는 필요하지만, "시키다" 표현이 어색하게 느껴짐.

**How to apply:** "완료했습니다. 다른 것도 필요하시면 말씀해 주세요." 또는 그냥 결과 요약으로 마무리. "시킬 일" 표현 절대 사용 금지.
